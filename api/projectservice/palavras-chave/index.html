    Palavras-chave

    🔑 Palavras-chave

    Listar Palavras-chave

    Visão Geral

    Retorna todas as palavras-chave cadastradas no sistema, com opção de filtro por nome.

    GET /keyword

    Tags: Keywords

    🔍 Parâmetros de Query

    Parâmetro Tipo Obrigatório Descrição Exemplo
    keyword string ❌ Nome da palavra-chave para filtro "pjbl"

    Exemplo de Requisição

    GET /keyword?keyword=pjbl

    ✅ Respostas

    Código Status Descrição
    200 ✅ Sucesso Lista de palavras-chave retornada

    Resposta de Sucesso (200)

    [
  {
    "id": 1,
    "keyword": "pjbl",
    "psp": "PSP1",
    "createdAt": "2024-01-15T10:00:00Z"
  },
  {
    "id": 2,
    "keyword": "agile",
    "psp": "PSP2",
    "createdAt": "2024-01-16T14:30:00Z"
  }
]

    Criar Palavra-chave

    Visão Geral

    Cadastra uma nova palavra-chave para utilização em projetos.

    POST /keyword

    Tags: Keywords

    📝 Corpo da Requisição

    Content-Type: application/json

    Campo Tipo Obrigatório Descrição Exemplo
    keyword string ✅ Nome da palavra-chave "pjbl"
    psp string ❌ Processo de Software Pessoal "PSP1"

    Exemplo de Requisição

    {
  "keyword": "pjbl",
  "psp": "PSP1"
}

    ✅ Respostas

    Código Status Descrição
    201 ✅ Criado Palavra-chave criada com sucesso
    400 ❌ Erro Dados inválidos
    409 ⚠️ Conflito Palavra-chave já existe

    Palavras-chave em Abstract

    Visão Geral

    Lista palavras-chave que estão relacionadas a projetos (presentes em abstracts).

    GET /keyword/abstract

    Tags: Keywords

    ✅ Respostas

    Código Status Descrição
    200 ✅ Sucesso Palavras-chave relacionadas retornadas

    Palavras-chave Disponíveis

    Visão Geral

    Lista palavras-chave que não estão associadas a nenhuma disciplina específica.

    GET /keyword/available

    Tags: Keywords

    ✅ Respostas

    Código Status Descrição
    200 ✅ Sucesso Palavras-chave disponíveis retornadas

    Editar Palavra-chave

    Visão Geral

    Modifica o valor de uma palavra-chave já cadastrada.

    PUT /keyword/available

    Tags: Keywords

    📝 Corpo da Requisição

    Content-Type: application/json

    Campo Tipo Obrigatório Descrição Exemplo
    keywordid integer ✅ ID da palavra-chave 1
    newKeyword string ✅ Novo valor da palavra-chave "newKeyword"

    Exemplo de Requisição

    {
  "keywordid": 1,
  "newKeyword": "newKeyword"
}

    ✅ Respostas

    Código Status Descrição
    200 ✅ Sucesso Palavra-chave atualizada
    404 ❌ Não encontrado Palavra-chave não existe

    Associar Disciplina à Palavra-chave

    Visão Geral

    Cria uma associação entre uma palavra-chave e uma disciplina.

    POST /keyword/subject

    Tags: Keywords

    📝 Corpo da Requisição

    Content-Type: application/json

    Campo Tipo Obrigatório Descrição Exemplo
    keywordid integer ✅ ID da palavra-chave 1
    subjectid integer ✅ ID da disciplina 2

    Exemplo de Requisição

    {
  "keywordid": 1,
  "subjectid": 2
}

    ✅ Respostas

    Código Status Descrição
    200 ✅ Sucesso Associação criada com sucesso
    404 ❌ Não encontrado Palavra-chave ou disciplina não existe

    Editar Associação Palavra-chave/Disciplina

    Visão Geral

    Modifica a associação existente entre uma palavra-chave e uma disciplina.

    PUT /keyword/subject

    Tags: Keywords

    📝 Corpo da Requisição

    Content-Type: application/json

    Campo Tipo Obrigatório Descrição Exemplo
    keywordid integer ✅ ID da palavra-chave 1
    subjectid integer ✅ ID da nova disciplina 2

    ✅ Respostas

    Código Status Descrição
    200 ✅ Sucesso Associação atualizada
    404 ❌ Não encontrado Palavra-chave ou disciplina não existe

    Exibir Disciplina por Palavra-chave

    Visão Geral

    Retorna a disciplina associada a uma palavra-chave específica.

    GET /keyword/subject/{keywordId}

    Tags: Keywords

    🎯 Parâmetros de Path

    Parâmetro Tipo Obrigatório Descrição Exemplo
    keywordId integer ✅ ID da palavra-chave 1

    ✅ Respostas

    Código Status Descrição
    200 ✅ Sucesso Disciplina relacionada retornada
    404 ❌ Não encontrado Palavra-chave não possui disciplina associada

    Excluir Palavra-chave

    Visão Geral

    Remove uma palavra-chave do sistema permanentemente.

    DELETE /keyword/{keywordId}

    Tags: Keywords

    🎯 Parâmetros de Path

    Parâmetro Tipo Obrigatório Descrição Exemplo
    keywordId integer ✅ ID da palavra-chave 1

    Atenção

    Esta operação remove permanentemente a palavra-chave e todas suas associações.

    ✅ Respostas

    Código Status Descrição
    200 ✅ Sucesso Palavra-chave excluída
    404 ❌ Não encontrado Palavra-chave não existe
    409 ⚠️ Conflito Palavra-chave em uso, não pode ser excluída
