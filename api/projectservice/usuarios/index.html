    Listar Subáreas

    Visão Geral

    Lista todas as subáreas cadastradas na plataforma com informações básicas.

    GET /subareas

    Tags: Areas

    Exemplo de Requisição

    GET /subareas

    ✅ Respostas

    Código Status Descrição
    200 ✅ Sucesso Lista de subáreas retornada

    Resposta de Sucesso (200)

  {
    "id": 1,
    "name": "Tecnologia da Informação",
    "description": "Área focada em TI",
    "areaId": 1,
    "areaName": "Ciências Exatas",
    "createdAt": "2024-01-15T10:00:00Z"
  }
]

    Usuários

    Listar Propostas do Usuário

    Visão Geral

    Lista todas as propostas de projeto criadas por um usuário específico.

    GET /userProposals/{userId}

    Tags: Projects

    🔍 Parâmetros de Path

    Parâmetro Tipo Obrigatório Descrição Exemplo
    userId integer ✅ ID do usuário 5

    Exemplo de Requisição

    GET /userProposals/5

    ✅ Respostas

    Código Status Descrição
    200 ✅ Sucesso Lista de propostas retornada
    404 ❌ Não encontrado Usuário não existe

    Resposta de Sucesso (200)

    [
  {
    "id": 1,
    "name": "Sistema de Gestão",
    "status": "created",
    "createdAt": "2024-01-15T10:00:00Z",
    "problem": "Necessidade de automatização",
    "expectedResult": "Maior eficiência"
  }
]

    Listar Turmas do Estudante

    Visão Geral

    Lista todas as turmas em que um estudante específico está matriculado com detalhes dos horários.

    GET /users/student/{userId}/class

    Tags: Users

    🔍 Parâmetros de Path

    Parâmetro Tipo Obrigatório Descrição Exemplo
    userId string ✅ ID do estudante "123"

    Exemplo de Requisição

    GET /users/student/123/class

    ✅ Respostas

    Código Status Descrição
    200 ✅ Sucesso Lista de turmas retornada
    400 ❌ Erro Requisição inválida

    Resposta de Sucesso (200)

    [
  {
    "classId": 1,
    "subjectId": 1,
    "classNumber": 1,
    "studentJoinMethod": "codigo",
    "accessCode": "ABCD1234",
    "semester": "1",
    "year": 2023,
    "active": true,
    "periods": [
      {
        "day": "segunda",
        "timeStart": "08:00",
        "timeFinish": "09:30"
      }
    ]
  }
]

    Resposta de Erro (400)

    {
  "error": "Error fetching classes"
}
